动作语句

写法：动作名 "执行对象" by 动作参数

对执行对象仿真动作。执行对象是用css选择器表达的唯一结果。input动作的参数

目前已经实现的动作包括：

  • mousein, 鼠标移入
  • mouseout, 鼠标移出
  • click, 左键单击
  • dblclick, 双击
  • rclick, 右键单击
  • input, 输入
  • select, 选择下拉菜单
示例代码： 
click "input#test"; //单机输入框
input "input#test" by "abced"; //在输入框里输入 abced
input "input#test" by [username]; //在输入框里输入 源自数据用例的username字段的值
input "input#test" by /[0-9]{4,8}/; //在输入框里输入 4~8位长的数字组成的字符串
dblclick "input#test"; //双击这个输入框
mousein "input#test"; //鼠标移入这个输入框

样例及其说明

下面是一个具体的案例，您可以通过这个样例完整了解LemonCase的编写规则。

	//需要先登陆后
	//测试创建n个测试项目
	jumpto '/lemonser';
	wait 2000;
	//跳转到我的项目界面
	click "div.selfCard > div.card";
	wait 2000;
	//记录一下当前有几个项目卡
	var projectCard=<#"div.card-container > div.card"/>;
	console projectCard;
	click "span.ico-addRound";
	wait 1000;
	input "div.grid > input.ng-pristine" by |[a-zA-Z]{8}|;
	input "textarea.ng-pristine" by "项目描述"+ |\d{4}|;
	click "div.grid > button:nth-child(5)";
	wait 1000;
	assert <#"div.card-container > div.card"/> > projectCard;
	console "创建项目成功";
